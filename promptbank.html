📘 How to Use Prompt Bank

  1. Transfer — deposit prompt baru. Pilih genre → isi judul & detail → Deposit. Balance naik +100 (gimik).
  2. Request — lihat daftar prompt available. Bisa search / filter genre. Klik Claim untuk mengambil.
  3. Transactions — semua aksi (submit/claim/complete) muncul seperti catatan bank.
  4. Claimed — prompt yang sudah di‑claim otomatis terkunci (abu‑abu & tombol disembunyikan).
  5. Posted — setelah karya dipost, organizer ubah status ke posted + link karya.
